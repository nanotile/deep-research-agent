 Here's the Claude Code CLI help output. Key highlights:
                                                         
  Basic Usage:                                           
  - claude - Start an interactive session                
  - claude "your prompt" - Start with a prompt           
  - claude -p "prompt" - Print mode (non-interactive,    
  useful for scripts/pipes)                              
                                                         
  Common Options:                                        
  - -c, --continue - Continue the most recent            
  conversation                                           
  - -r, --resume [id] - Resume a specific session        
  - --model <model> - Choose model (e.g., sonnet, opus)  
  - --allowedTools - Restrict which tools can be used    
  - --system-prompt - Custom system prompt               
                                                         
  Commands:                                              
  - claude doctor - Check auto-updater health            
  - claude install - Install/update Claude Code          
  - claude mcp - Configure MCP servers                   
  - claude plugin - Manage plugins                       
  - claude update - Check for updates
